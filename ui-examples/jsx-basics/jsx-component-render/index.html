Renderização de Componentes do JSX

Explicando os conceitos do código JSX

Este exemplo demonstra a renderização de componentes em JSX. A funçãorenderCircle é responsável por criar um elemento reutilizável para renderizar círculos numerados.

  • Função de renderização → A função renderCirclecria um componente de círculo estilizado.
  • Uso de componentes reutilizáveis → Chamamos a função várias vezes para gerar os elementos dentro da grade.
  • Grid Layout → Utilizamos grid-cols-3 para organizar os elementos em três colunas.

    function renderCircle(text: string) {
      return (
        <div className="flex justify-center items-center h-12 w-12 rounded-full 
          bg-blue-500 text-white text-sm">
          {text}
        </div>
      );
    }
    
    return (
      <div className="grid grid-cols-3 gap-4">
        {renderCircle("#01")}
        {renderCircle("#02")}
        {renderCircle("#03")}
        {renderCircle("#04")}
        {renderCircle("#05")}
        {renderCircle("#06")}
      </div>
    );

A reutilização de componentes torna o código mais modular e fácil de manter. Podemos alterar a função renderCircle sem precisar modificar diretamente cada instância dos elementos renderizados.
